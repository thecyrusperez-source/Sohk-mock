# Roster photos — drop rights-cleared, first-party portraits here

The site wires each athlete/coach card to a file in this folder. Until a real,
**rights-cleared** photo is added, the card shows a chrome monogram fallback.

Per the personnel research: Instagram / BoxRec / Getty images are NOT cleared for
commercial web use. Use first-party photos the gym owns or has written permission
to publish. Recommended specs: portrait/4:5, min 800×1000px, JPG.
