# Hero video — drop the looping background clip here

Add a muted, looping fight/training clip named `hero.mp4` (and optionally
`hero.webm`). The hero `<video>` is already wired to these paths and treated to
black & white via CSS. Until a file exists, a cinematic CSS backdrop shows.
Recommended: 1920×1080+, H.264 MP4, ~10–25s seamless loop, no audio track.
